Functional Requirements

Group Members: Baldwin, Chunda, Ian

Subject: Note Taking App

  • Input Text
    • Ability for user to type text via normal keyboard procedures
    • Possible ability to attach a date/time and/or alert to that text
  • Save Notes(string of text)
    • Ability to save the text with one button press, to be displayed, edited or deleted
  • Display notes in a list
    • Ability for user to see list of all saved notes
    • (Display only the first X number of characters, then expand?)
    • Notes should be ordered newest to oldest (top to bottom)
  • Delete note in list
    • Ability to notes in list of notes
  • Edit previously saved note
    • Ability to edit notes in list of notes
  • Long Press Menu or button menu
    • Buttons next to listed notes could be used to edit or delete.
    • (How does this function work on Android?) Small pop-up on long-press with edit and delete buttons
    • This can be one or the other.
